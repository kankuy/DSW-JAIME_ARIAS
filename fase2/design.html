Etiquetas basicas de Html

Las etiquetas son la estructura básica del HTML. Estas etiquetas o tags se componen y contienen otras propiedades, como son los atributos y el contenido.
HTML define un total de 91 etiquetas, de las cuales 10 se consideran obsoletas. Sin embargo, una etiqueta por
sí sola a veces no contiene la suficiente información para estar completamente definida. Para ello
contamos con los atributos: pares nombre-valor separados por "=" y escritos en la etiqueta inicial de
un elemento después del nombre del elemento. El valor puede estar encerrado entre "comillas dobles" o
'simples'. Existen, también, algunos atributos que afectan al elemento por su presencia en la etiqueta de inicio, como puede ser el atributo ismap para el elemento.

Atributos y valores

1 Añadimos una nueva etiqueta de tipo imagen. Es decir, incluímos la etiqueta "< img>".
2 Añadimos un atributo a esa etiqueta. El nombre del atributo que añadimos es “src” que corresponde a
“source”, que traducido significa “fuente”. Con este atributo lo que especificamos fue la fuente de la imagen.
3 Especificamos el valor del atributo source. Le dimos valor a ese atributo. En concreto le dimos el valor
“imagen.png” (o “imagen.jpg”).
