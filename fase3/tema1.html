ETIQUETAS BASICAS DE HTML?

Etiquetas HTML

las etiquetas de HTML, estan delimitadas por un inicio y un final de casa elemento que insertamos.
lo que se encuentra a dentro de la etiqueta (el contenido) es lo que estamos formateando.
para insertar una etiqueta se necesita escribir:

body para el contenido
head para información sobre el documento
div división dentro del contenido
a para enlaces
strong para poner el texto en negrita
br para saltos de línea
H1…H6 para títulos dentro del contenido
img para añadir imágenes al documento
ol> para listas ordenadas, ul> para listas desordenadas, li> para elementos dentro de la lista p> para parágrafos
span> para estilos de una parte del texto
